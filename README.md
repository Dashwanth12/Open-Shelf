# OpenShelf 📚

A minimalist digital archive for book discovery. Developed using React and Supabase, this platform offers a curated UI/UX for searching and organizing your personal library via the Open Library API.

## ✨ Features
* **Minimalist UI:** A sophisticated, editorial design focusing on typography and whitespace.
* **Book Discovery:** Real-time search powered by the **Open Library API**.
* **Secure Authentication:** Passwordless login using **Supabase Email OTP**.
* **Personal Archive:** Organize your books into a personalized shelf.

## 🛠️ Tech Stack
* **Frontend:** React (Vite)
* **Backend:** Supabase
* **Styling:** CSS-in-JS / Styled-Components
* **API:** Open Library

## 🚀 Getting Started
1. **Clone the repo:** `git clone https://github.com/Dashwanth12/Open-Shelf.git`
2. **Install:** `npm install`
3. **Environment:** Add your Supabase keys to a `.env` file.
4. **Run:** `npm run dev`
